# CodeAlpha URL Shortener

A simple URL Shortener built using Flask and SQLite.

## Features
- Shortens long URLs
- Stores URLs in SQLite database
- Redirects short URLs to original URLs

## Technologies Used
- Python
- Flask
- SQLite

## How to Run
1. Install Flask:
   pip install flask

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
